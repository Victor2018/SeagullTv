# 项目介绍：
	这是一款MaterialDesign设计风格视频播放器包含央视、卫视、地方、体育、卡通等高清直播以及youtube，vimeo视频爬虫及播放。直播，youtube，vimeo资源来自互联网，本应用只做学习使用，请勿商用！
